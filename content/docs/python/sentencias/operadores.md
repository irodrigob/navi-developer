# Operadores

## Aritméticos

Los operadores son los habituales:
* Sumar: +
* Resta: -
* División: /
* Multiplicación: *
* Elevar a la potencia: **
* Obtener el resto: %

Ejemplos:

```tpl
d = a * b
print("Resultado", d)
d = ( a + b ) / a
print("Resultado:", d)
```

Resultado:

```
Resultado 416.15999999999997
Resultado: 3.89
```
